# Contributing to Agentic System

Thank you for your interest in contributing to the Agentic System project!

## Ways to Contribute

### 1. Independent Verification

The most valuable contribution is running independent verification:

1. Follow the [Verification Guide](VERIFICATION.md)
2. Run AVIR or manual benchmarks
3. Submit a Verification Report via GitHub Issues

### 2. Bug Reports

If you encounter issues during installation or verification:

1. Check existing issues first
2. Include your environment details (OS, RAM, container runtime)
3. Attach relevant logs
4. Describe steps to reproduce

### 3. Documentation Improvements

- Fix typos or unclear instructions
- Add examples for edge cases
- Improve troubleshooting guides

### 4. Benchmark Contributions

- Test on different hardware configurations
- Report performance on cloud VMs (AWS, GCP, Azure)
- Identify platform-specific issues

## Code of Conduct

- Be respectful and constructive
- Focus on technical merit
- Help others succeed in verification

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a PR with clear description
5. Respond to review feedback

## Questions?

Open a Discussion on GitHub or reach out via Issues.
