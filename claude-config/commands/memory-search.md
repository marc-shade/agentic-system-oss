# Search the knowledge graph

Search persistent memory for relevant information from past sessions.

## What can be found
- Previous learnings and insights
- Project context and decisions
- Code patterns and solutions
- Technical knowledge

## Search strategies
1. **Semantic search**: Natural language queries
2. **Type filter**: Search specific memory types
3. **Recency**: Recent vs historical items

---

Search memory for: $ARGUMENTS

Use mcp__enhanced-memory__search_nodes with:
- The query text
- limit: 10 (or more if comprehensive search needed)

Then summarize the relevant findings.
